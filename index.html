For user interaction the orbit control was added in order to allow a user to see all sides of the scene by moving around the camera point view. Movements of camera point view can be done by sliding the mouse.

The Down arrow key is for moving the figure forward, the Up arrow key is for moving backward. The Left arrow key is for moving the figure to the left side, and the Right arrow key is for the right side.
